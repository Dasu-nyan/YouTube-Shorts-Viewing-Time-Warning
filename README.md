# YouTube Shorts Timer

このChrome拡張機能は、YouTube Shortsを指定した時間以上視聴した場合に通知を表示して警告します。
YouTube Shortsを見すぎてしまった経験から、視聴時間をうまくコントロールしたいと考え、作成しました。
決してYouTube Shortsを攻撃するつもりはなく、視聴者が他の活動も楽しみながら時間を有効に使えるようサポートすることが目的です。
同じ悩みを持つ方々に少しでも役立てば幸いです。

## 機能 (Features)
- YouTube Shorts の視聴時間にカスタム制限を設定できます（時、分、秒単位）。
  - デフォルト設定は10分です
- 制限時間を超えると通知が表示されます。
- ユーザーフレンドリーなインターフェイスで簡単に設定変更が可能。
- 現在の視聴時間をリアルタイムで表示します。

## インストール手順 (Installation)
1. このリポジトリをダウンロードまたはクローンします。
2. Chromeを開き、`chrome://extensions`にアクセスします。
3. **開発者モード**を有効にします（右上）。
4. **パッケージ化されていない拡張機能を読み込む**をクリックし、プロジェクトフォルダを選択します。
5. 拡張機能が有効になり、使用可能になります。
